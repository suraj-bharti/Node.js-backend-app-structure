# NodeJs-Startup-App
A development structure for creating a Node backend for your need.

Just clone this repository and start your developemnt and make changes as per your requirement.

## Install App
```
npm install
```
## Start App (dev)
```
npm run dev
```
## Start App (production)
```
npm run start
```
